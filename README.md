# Chip Sort

GUI-based application to teach users the inter-workings of various sorting algorithms.

## Getting Started

### Dependencies

* Java JDK
* JavaFX
* Windows, Mac, Linux


### Executing program

* Install Java JDK
* Open command Prompt
* Type: "Javac ChipSort.java"
* Then, Type: "Java ChipSort"
